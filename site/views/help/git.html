    commit

    git è un repository distribuito. per committare una versione bisogna prima aggiungere gli eventuali nuovi file con git add, committare sul repository locale con git commit, e quindi sincronizzare il repository locale con il repository di origine sul branch master con git push origin master.

    repository su github.com

    github.com fornisce dei repository git gratuiti (e pubblici) o a pagamento (e privati). Per scaricare un progetto da github è possibile utilizzare il comando git clone

    Per poter committare senza che vengano chiesti ogni volta utenza e password, e' possibile modificare il file .git/config aggiungendo nome utente e password nell'url del repository, utilizzando il formato username:password@github.com
